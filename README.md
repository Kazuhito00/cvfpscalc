# cvfpscalc
 cvfpscalcはOpenCVのgetTickFrequency()を利用したFPS計測クラスです。

# Requirement
 
* OpenCV 3.4.2
 
# Installation
 
利用したいPythonプログラムと同階層にcvfpscalcディレクトリをコピーしてください。
 
# Usage
 
サンプルの実行方法は以下です。Webカメラが接続されている必要があります。
 
```bash
python sample.py
```
